POPL 2018 AEC Review material

Instructions

The tool described in the paper, QuickChick, is a publicly available tool for the Coq proof assistant. For the artifact evaluation we have frozen the "packaging" branch here.

  • We added an opam installer that should take care of the installation procedure. Just do the following from the top directory:

    opam install quickchick

  • Alternatively, you could use make for QuickChick itself; instructions appear at the README. In short:

    # To get ssreflect, you need to add the Coq opam repository (if you haven't already)

    opam repo add coq-released https://coq.inria.fr/opam/released
    opam update
    opam install coq-mathcomp-ssreflect

    # Then:
    make && make install

  • The basic functionality of QuickChick is described in the form of a module signature in BasicInterface.v.

  • An extended tutorial for QuickChick, including a mini-tutorial on typeclasses appeared at the DeepSpec summer school 2017 and is freely available.

  • The derivation procedures described in the paper are implemented as toplevel vernacular commands. For simple examples, you can look at DependentTest.v and at dependentProofs.v in the examples folder.

  • The case study on Software Foundations can be found under the folder examples/sf.
